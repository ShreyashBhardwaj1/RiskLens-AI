# RiskLens-AI: Credit Risk Prediction & Lending Intelligence Platform

## Overview
RiskLens-AI is an AI-powered credit risk prediction system designed to help financial institutions evaluate loan applicants, predict default probability, and support intelligent lending decisions.

This project demonstrates an end-to-end machine learning pipeline including:

- Data preprocessing
- Feature engineering
- Model training and evaluation
- Risk prediction
- Model explainability (SHAP)
- Deployment-ready architecture

---

## Problem Statement
Financial institutions face significant losses due to loan defaults. This project builds a machine learning model to:

- Predict loan default probability
- Identify key risk factors
- Support risk-based lending decisions
